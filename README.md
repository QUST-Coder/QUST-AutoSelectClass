## 适用于青岛科技大学新教务自动抢课
### 使用方法
- 浏览器打开到选课页面（注意是通识选修课页面）
- 记得点击“加载更多”把所有网课都显示出来
- 打开开发者工具，选择console
- 将代码粘贴到console中，回车
- enjoy！
